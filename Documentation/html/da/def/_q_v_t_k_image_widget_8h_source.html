US Probe Calibration with LSQRREcipes  1.0
This classes implement the LSQRRecipes methods to calibrate an Ultra Sound probe using a cross wire phantom. The classes are programmed in c++ using the LSQRRecipes, ITK, VTK and Qt libraries.
